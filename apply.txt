======================================================================
一、项目简介
======================================================================

CodeRev Agent 是一个基于多智能体协同架构的 AI 代码审查 CLI 工具。
它利用大模型的长上下文和推理能力，在代码提交前自动进行多维度的深度审
查，帮助开发者在早期发现安全漏洞、性能隐患、逻辑缺陷和风格问题。

与传统单次审查不同，CodeRev 将审查任务分解为四个专业维度，由四个独立
的 Agent 并行执行，每个 Agent 专注于自己的领域，最终聚合为一份结构化
的审查报告。

======================================================================
二、核心技术架构
======================================================================

1. 多智能体协同审查
   同时启动四个专用 Agent 并行工作：
   - 安全审查 Agent：SQL注入、命令注入、XSS、硬编码密钥、不安全反序列化
   - 性能审查 Agent：N+1查询、循环内IO、缺失生成器、重复计算、内存泄漏
   - 逻辑审查 Agent：空值检查、越界、异常吞没、除零、不可达代码
   - 风格审查 Agent：长函数、深层嵌套、命名规范、类型注解、DRY原则

2. 长链路推理分析
   利用 Python AST 解析源代码，提取函数定义、导入关系和调用依赖，自动
   构建跨文件的调用图（Call Graph）。每个 Agent 沿调用链逐层追踪数据流
   和异常传播路径，而非孤立地审查单个函数。

3. 结构化报告输出
   审查结果按严重程度（Critical/Warning/Info）分级，每条问题附精确的
   文件路径和行号，以及可操作的修复建议。支持 Markdown 和 JSON 两种输
   出格式，可无缝集成到 CI/CD 流水线。

======================================================================
三、技术栈
======================================================================

- 后端模型：MiMo API（OpenAI 兼容接口），利用 100万 token 上下文窗口
  支撑多文件代码分析和大规模调用图追踪
- 代码解析：Python AST 静态分析
- CLI 框架：Python + argparse + Rich 终端美化
- 并发调度：ThreadPoolExecutor 多线程并行 Agent 调用
- 版本集成：Git diff / staged changes 审查

======================================================================
四、使用场景
======================================================================

- Pre-commit 审查：git commit 前运行 coderev diff，自动拦截高风险代码
- CI 集成：在 GitHub Actions 中自动审查每个 PR 的变更
- 遗留代码审查：coderev review src/ 一键扫描整个项目
- 新人 Onboarding：帮助新成员理解代码质量基线

======================================================================
五、MiMo API 使用计划
======================================================================

1. 利用 MiMo 100万 token 上下文窗口，一次加载多个源文件和完整调用图，
   避免传统分块审查的上下文断裂问题
2. 四个 Agent 并发调用 MiMo API，利用低延迟特性实现快速审查反馈
3. 在 system prompt 中内置专业审查知识库（OWASP Top 10、常见性能反
   模式、Python 最佳实践），每次审查自动注入，无需额外训练
4. 后期计划支持 MiMo 的多模态能力，对架构图、流程图进行审查
